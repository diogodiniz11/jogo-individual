# jogo-individual

## Sobre o Jogo
Você é um círculo vermelho preso em uma arena com um triângulo verde assassino.
Sua missão: sobreviver por 1 minuto e coletar o máximo de moedas possível antes
que o triângulo te alcance!

## Como Jogar
- **WASD** ou **setas do teclado** para mover o círculo
- Fuja do triângulo verde — ele te persegue!
- Colete as moedas espalhadas pelo mapa
- Sobreviva por **1 minuto** para vencer

## Objetivo
Chegue ao fim do cronômetro com o maior score possível!
